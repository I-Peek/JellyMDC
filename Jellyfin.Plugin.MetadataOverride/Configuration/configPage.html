JellyMDC

Choose which fields should be protected, then press Capture snapshot now. JellyMDC stores the current values of the selected fields and re-applies them after every refresh – matched by TMDB/IMDB id, so a file rename does not break it. Run the capture again whenever you have changed something in Jellyfin that you want to keep.

Default rules

These apply to your whole library. Tick the fields that should be kept for each type.

Exceptions

Individual titles that should use their own field selection instead of the default above. Search for a title, add it, then tick its fields. Episodes always follow the series default.
